Help Button

Opens the page you are currently reading.

Reset Buzzers Button

Resets the podium buzzers so that another player may buzz in. Press if a player has buzzed in accidentaly or has buzzed in and changed their mind.

Next Button

Goes next or something.

Final Round Indicator

When illuminated in yellow, indicates that the current round is the last round of the game. This is either happens when running out of available questions, or when the event is nearly over.

Control Panel Button

Opens and closes the control panel (more information below).

Podium Controls

Add/Sub:
Manually add or subtract a point from the selected player. Note, points are awarded automatically

Rename:
Change the name of the selected player. If a player with a long display name is playing, you can use the rename button to shorten the name. (Tip: if a player with a guest account is playing, you can ask them for their real name or nickname and rename them.)

Kick:
Remove the selected player from the game.
